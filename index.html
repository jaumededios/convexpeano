03 / What is happening?

A finite shadow of an infinite construction.

A

The traversal

A Hilbert-style recursion orders finer and finer square cells along [0,1]. Increasing the resolution reveals the familiar space-filling limit.

B

The selected image

Your interval becomes a contiguous run of recursive cells. The orange area is that finite image; the blue boundary is its live convex envelope.

C

The real monstrosity

Paszkiewicz constructs nested ordered populations of convex sets so every consecutive union is already convex—no envelope required.

Honest caveat

This is an interactive analogy, not a numerical implementation of the paper’s proof. It swaps the proof’s “populations of souls” for a fast recursive traversal, then makes the convexity visible explicitly.

“There exists a continuous surjection f : [0,1] → T for which the image of any interval is a convex set.”
— Adam Paszkiewicz, The Convex Peano Curve Does Exist
